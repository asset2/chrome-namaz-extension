# Следующий намаз — Chrome Extension

## Что делает
- Загружает расписание намазов с страницы Sajda.
- Поддерживает выбор города: Алматы, Астана, Шымкент.
- Показывает ближайший следующий намаз и обратный отсчёт.
- По кнопке "Обновить" перечитывает данные.
- Интерфейс в тёмной теме.
- Если сайт недоступен, показывает последние сохранённые данные (если они есть).

Источник:
https://sajda.com/ru/prayer-times/kazakhstan/almaty/1526384

## Установка в Chrome
1. Откройте `chrome://extensions/`
2. Включите "Режим разработчика".
3. Нажмите "Загрузить распакованное расширение".
4. Выберите папку `chrome-namaz-extension`.
5. Закрепите расширение и откройте popup.

## Использование
1. Выберите город в правом верхнем углу popup.
2. Расширение автоматически сохранит выбор.
3. При следующем открытии будет выбран последний город.
